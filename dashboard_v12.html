V11 Hybrid 3-Way — Detalhe

Para 14 traits fracos (baixa h2 ou PA negativa), V11 combina 4 sinais via Ridge:

final = Ridge.predict([V11_pred, pred_2stage, pred_anchored, PA])

V11_pred = PA + deviation (modelo padrao)
pred_2stage = sign(classificador) x magnitude(regressor) x confidence
pred_anchored = sire_profile_mean x 0.85 + MGS x 0.15 (se n>=5 filhas)
PA = Sire/2 + MGS/4 + MMGS/8

confidence = 0.6 x sign_confidence + 0.4 x min(n_daughters / 30, 1.0)

Top Group & Recall@K

Estrategia de selecao: selecionar top K% predito para capturar o top 5% real

Como Funciona

Em um rebanho de 1.000 animais, os top 5% = 50 animais. Se selecionarmos os top 10% preditos (100 animais), quantos dos 50 realmente melhores estao entre esses 100? Essa e a metrica de Recall@K.

Recall@K por Trait — V11

TraitTop 5% AccRecall@10%Recall@15%Recall@20%Recall@25%
Recall@10% = "Se seleciono os 10% melhores preditos, capturo X% dos 5% realmente melhores". Validado no Ethan (4.156 animais).

Estrategia de Producao

Selecionar top 10% predito: Recall medio = 67% (captura ~2/3 dos top 5% reais)
Selecionar top 15% predito: Recall medio = 78% (captura ~3/4 dos top 5% reais)

Exemplo FAT: top 10% predito captura 85% dos top 5% reais
Exemplo DPR: top 10% predito captura 79% dos top 5% reais

Calibracao V12 (2-Fatores + Daughter)

Parametros c0, c1, c2 (2F) e d0, d1, d2, var_expansion (Daughter) — Descontinuada na producao

Formulas de Calibracao

Calibracao 2-Fatores:
PTA_2F = c0 + c1 x PA + c2 x ML_deviation

Calibracao Daughter:
raw = d0 + d1 x PA + d2 x ML_deviation
PTA_DAU = PA + (raw - PA) x var_expansion

Parametros por Trait

Traitc0c1 (PA)c2 (ML)d0d1d2VarExp

Formulas & Indices

Parent Average, indices compostos, pesos de treinamento

Parent Average (PA)

PA = Sire_PTA / 2 + MGS_PTA / 4 + MMGS_PTA / 8

Se MGS ausente: PA = Sire_PTA / 2
Se MMGS ausente: PA = Sire_PTA / 2 + MGS_PTA / 4

Fertility Index (FI)

FI = 0.4 x DPR + 0.4 x CCR + 0.1 x HCR + 0.1 x EFC  (CDCB 2024)

Net Merit (NM$)

NM$ = -0.089 x MILK + 1.20 x FAT + 1.86 x PRO + 28.0 x PL - 176.0 x SCS
     + 11.6 x DPR + 3.8 x HCR + 5.2 x CCR + 14.0 x LIV - 3.4 x MAST
     + 17.5 x PTAT + 14.8 x UDC + 5.2 x FLC

Cheese Merit (CM$)

CM$ = -0.15 x MILK + 0.85 x FAT + 2.60 x PRO + 26.0 x PL - 165.0 x SCS
     + 11.0 x DPR + 3.5 x HCR + 5.0 x CCR + 13.0 x LIV - 3.2 x MAST
     + 19.0 x PTAT + 16.0 x UDC + 5.5 x FLC

Pesos de Treinamento (V12)

w_gen = 0.5 + 0.5 x min((DOB_year - 1990) / 30, 1.0)
w_rel = Reliability / 100
w_final = w_gen x w_rel
Filhas: w = 1.0 (fixo)
Filtros BTB:
- Reliability >= 75%
- DOB >= 2005
- Sire stack parseado
